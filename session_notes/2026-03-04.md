## Priority 3 Implementation: Model Selection Section (Evening)

**Goal:** Add Section 5.2 on model selection for extrapolation with time-series cross-validation

**Status:** ✅ COMPLETED

### Implementation

Added Section 5.2 (~3 pages) with comprehensive model selection framework:

1. **Problem Statement**
   - Why AIC/BIC fail for extrapolation (measure in-sample fit, not future prediction)
   - Need for extrapolation-specific validation

2. **Time-Series Cross-Validation Procedure**
   - Training/test split: fit on t ≤ p-h, predict on held-out t ∈ {p-h+1,...,p}
   - MSPE metric: Mean squared prediction error on held-out periods
   - Multiple horizons: h=1,2,3 to test different extrapolation distances
   - Selection rule: minimize average MSPE across horizons

3. **Coverage-Based Selection**
   - Complementary criterion: 95% CI coverage on held-out periods
   - Identifies models underestimating uncertainty

4. **Practical 5-Step Workflow**
   - Specify candidate models (linear, quadratic, spline)
   - Run time-series CV → compute MSPE and coverage
   - Select best OR model-average with exponential weights
   - Report sensitivity: top 2-3 models
   - Plot diagnostics: fitted curves extended to p+1

5. **Honest Limitations**
   - CV tests extrapolation within observed window, not to genuinely future P_{p+1}
   - Assumes stable DGP across t ≤ p
   - May fail under regime change
   - MSPE averages over groups (may mask heterogeneity)

6. **Post-Selection Inference Note**
   - Acknowledges CV introduces selection uncertainty
   - References selective inference literature (Tibshirani et al. 2016, Chen et al. 2021)
   - "We leave formal treatment to future work, but CV provides principled empirical guidance"

### Key Innovation

**This is novel for DiD extrapolation literature:** Instead of vague "try different models," we provide:
- Rigorous, actionable validation framework
- Directly tests extrapolation (not in-sample fit)
- Mimics the actual prediction task
- Provides concrete workflow researchers can follow

### Citations Added

- Tibshirani et al. (2016) - Exact post-selection inference
- Chen et al. (2021) - Valid inference after model selection
